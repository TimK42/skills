# OpenClaw Skills

Curated skills for multi-agent fix loops, backlog management, and UI/UX audits.

## Skills

| Skill | Description |
|-------|-------------|
| **backlog-md** | File-based task manager. Manage deferred tasks via `memory/backlog.md`. No GitHub required. |
| **issue2merge** | Multi-agent fix loop that drives a GitHub issue from triage to merge-ready PR. Manager coordinates parallel Fix, Review, and Watch sub-agents. |
| **uiux-audit** | Browser-based UI/UX audit covering accessibility, responsive design, i18n, PWA, form UX, and visual consistency. |

## Installation

```bash
npx skills@latest add TimK42/skills
```

Then pick the skills you want to install.
